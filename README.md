# Aayush Mehta — Portfolio

Personal portfolio website built with React, Vite, and Tailwind CSS.

**Live:** https://aayush-mehta-0924.vercel.app

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Framer Motion
- tsParticles

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:5173 in your browser.

## Project Structure

```
src/
  assets/
    images/
      profile/     # Personal photos
      projects/    # Project thumbnails
    icons/
  components/
    layout/        # Navbar, Footer, Hamburger, Logo
    sections/      # Hero, About, Experience, Projects, Contact, ...
    ui/            # Reusable components (MagneticButton, Terminal, ...)
  constants/       # Site content and data
  context/         # Theme and Sound context
```

## Build

```bash
npm run build
```
